# Lettudy!

## 📍 프로젝트 개요

### Let’s Study! Lettudy!
Lettudy는 스터디를 하며 발생하는 번거로운 일들을 줄여주고, 동기부여를 도와주는 스터디 관리 서비스입니다 <br />

### 개발기간
2024.07.22 ~ 2024.08.30

## 📍 개발파트

**강정윤**

- 회원가입 API 구현
- 일정(캘린더) API 구현
- 채팅 API 구현
- 채팅 클라이언트 연동
- 자료모음 API 구현

**박은지**

- 스터디방 API 구현
- 공지 API 구현
- 스터디방 멤버 관리 API 구현
- 웹 푸시 알람 구현
- 노트작성 API 구현
